Features

💬

AI Chat in Terminal

Interactive AI assistant right in your terminal. Ask questions, get code, fix bugs.

📂

Read & Write Files

AI can read your code, understand context, and create or edit files directly.

⚡

Execute Commands

Run shell commands from within the chat. Build, test, deploy — all from one place.

🔍

Search Codebase

Search across your entire project for functions, variables, and patterns.

🌐

Web Dashboard

Built-in web UI for managing settings, bot connections, and monitoring status.

📱

Telegram Bot

Connect your Telegram bot and chat with Zaifi Claw from your phone.

🔒

Secure

API keys stored locally. No telemetry. No data collection. 100% private.

💰

Free

Powered by Google Gemini's free tier. No VPS, no subscriptions, no hidden costs.

Quick Start

1

Install

npm install -g zaificlaw
2

Setup API Key

zaificlaw --setup

Get your free Gemini key from aistudio.google.com

3

Start Coding

cd my-project && zaificlaw
